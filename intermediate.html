Compare Economic Indicators in FRED

Goal: Create a graph that compares inflation and unemployment over time.

Watch the tutorial

Steps

  1. Go to the FRED website.
  2. In the search box type consumer price index.
  3. Select Consumer Price Index for All Urban Consumers (CPIAUCSL).
  4. Click the Add Line button above the graph.
  5. Search for unemployment rate.
  6. Select Civilian Unemployment Rate (UNRATE) to add it to the graph.
  7. Adjust the time range to compare trends over the last 10–20 years.
  8. Observe how inflation and unemployment move over time.
